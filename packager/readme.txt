Anyframe ODEN, release 2.5.0 (2013.04.05)
---------------------------------------------------

1. Anyframe ODEN 소개
Anyframe Open Deployment ENvironment은 CI 환경을 통해 빌드된 어플리케이션 컴포넌트 및 각종 설정파일, 웹파일 등을 원하는 배포대상서버에 편리하게 배포할 수 있도록 하는 배포관리 툴이다.
 
 
* Anyframe 포탈 사이트 : http://www.anyframejava.org 


2. 배포 파일 구조(zip)

  Anyframe-Oden-[version].zip
   - core  
	- bin : Anyframe ODEN의 실행 스크립트
	- bundle : Anyframe ODEN 및 라이브러리
	- conf : Anyframe ODEN의 설정 파일
   - admin : ODEN Admin 파일 및 실행 스크립트
   - manual : ODEN의 사용을 위한 사용자가이드 매뉴얼
   - readme.txt : Anyframe ODEN 소개 및 기본 사항


3. 라이선스 정책
Anyframe ODEN은 라이센스 정책으로 Anyframe Java Comunity Edition을 채택하며, 라이센스 정책은 license.txt 파일을 참고하면 된다.
